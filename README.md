# api-nodejs2023

<h3>Dependencies</h3>
- yarn add express
- yarn add sequelize
  * acesso ao banco de dados
- yarn add sequelize-cli -D
  * linha de comando que ajuda criar codigos mais rapidos, migrations, models
- yarn add nodemon -D
- yarn add eslint -D

<h3>Run project</h3>
- yarn dev - development environment
- yarn start - production environment
